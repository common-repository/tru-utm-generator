Generate UTM links

== Description ==

UTM Generator plugin allows you to easily add campaign parameters to URLs so you can track your Custom Campaigns for your website. These campaign parameters can be Campaign name, Source, Medium etc. that will be included into auto generated URL when you press Generate button.

You can add parameters and values to your URLs manually, but UTM Generator provides you the platform where you just need to enter desired parameters and it will auto-generate campaign URL for you. 


== Installation ==

1. Upload the `tru-utm-generator` folder to the directory `/wp-content/plugins/`.
2. Activate the plugin using the 'Plugins' menu in WordPress.

== Frequently asked questions ==

## How to use

1. First Activate Plugin.
2. Go to settings Menu and save settings.
3. Paste shortcode [tru_utm_generator] in your pages or posts to generate utm links
